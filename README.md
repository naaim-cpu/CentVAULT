# Claims Desk public website

This is the standalone public-hosting version of the Claims Desk.

## Run locally

The site loads JSON and document files, so serve this folder with a small local web server instead of opening `index.html` directly.

## Free hosting

Upload the complete folder to a static host such as GitHub Pages. Keep the `data` and `vendor` folders beside `index.html`.

No hosting provider can guarantee a free address forever. A `github.io` subdomain remains available while the GitHub account, repository and Pages service remain active.

## Privacy warning

This package contains the claims knowledge source files. Publishing it from a public repository makes those files publicly accessible, even when the website does not link to every file directly.
